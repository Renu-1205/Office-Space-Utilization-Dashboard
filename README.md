# Office Space Utilization Dashboard

## Project Overview

**Business Problem:** Organizations waste millions on underutilized office space. Without usage data, facility managers cannot optimize layouts or reduce costs.

**Solution:** An interactive Power BI dashboard that tracks space utilization by space type, department, and time patterns.

**Impact:** Enables data-driven decisions to reduce real estate costs by **15-25%**

---

## Key Insights from Dashboard

| Space Type | Utilization % | Cost per Occupant |
|------------|---------------|-------------------|
| Collaboration area | 15% | 15 |
| Meeting room | 12% | 18 |
| Workstation | 10% | 19 |
| Private office | 10% | 100 |
| Lounge | 8% | 18 |

**Key Finding:** Collaboration areas and meeting rooms show LOWEST utilization despite HIGHER costs.

---

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query

## Data Source
- 200 records of office space utilization
- 10 departments
- 5 space types
- Multiple office locations
